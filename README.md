# Weather Forecast App

## Overview
The Weather Forecast App is a simple and user-friendly application that provides real-time weather information for any location. It is built using the OpenWeather API to fetch accurate and up-to-date weather data, including temperature, weather conditions, humidity, and wind speed.

## Features
- Search weather details by city name  
- Real-time temperature and weather conditions  
- Displays humidity, wind speed, and atmospheric details  
- Clean and responsive user interface  
- Fast and reliable API integration  

## Technologies Used
- Frontend: HTML, CSS, JavaScript  
- API: OpenWeather API
